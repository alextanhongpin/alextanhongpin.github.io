Basic CSS3 Tutorial

Styling with CSS

You can style your page with CSS. CSS stands for Cascading Style Sheet, and the 3 means the version of the CSS used.

To start styling your page, first add the style element in between the head tags and before the title. The reason is simple - the page loads from top to bottom, and we want the style element to be loaded first before the body so that when the body load we can see the styled page immediately.

<!DOCTYPE html>
<html lang='en'>
    <head>
	<style>
	/*comment: style goes in here*/
	<style>
	
        <title>title goes here</title>
    </head>
    <body>
    </body>
</html>

You can see that there's a comment in between the style tags. You can comment a line of code by placing them between a /* and */ tag. Commenting is especially useful when you want to test codes. Basically what is commented will be inactive and will not be showed on a page.

Let us make change the background color of the body. We can do this by calling the elements in the style tags and placing the corresponding syle-options in between two curly brackets. 

<!DOCTYPE html>
<html lang='en'>
    <head>
	<style>
	/*#dd = light grey*/
	body{
		background: #ddd; 
	}
	<style>
	
        <title>title goes here</title>
    </head>
    <body>
    </body>
</html>

And the page will turn grey. What does hash d d d means? In HTML, we specify the color by RGB standards (red, green, and blue). There are 17 levels of color tone, starting from the lightest f, followed by e,d,c,b,a,9,8,7,6,5,4,3,2,1 and the darkest 0.

For example, if we want red color, then we specify #f00, which literally means FULL Red, ZERO Green and Zero Blue. To lighten the color, just increase the level of green and blue proportionally. Eg, #f22 (lighter red), #f44 (even lighter).

To get the color green, the code will be #0f0. For the color blue, it's...yup you guessed it correctly! It's #00f. You will realise the using such high contrast colors won't do your webpage any good. I prefer neutral colors, so I usually pick any colors in the range 8-d. Colors like #fdd or #dfd looks more pleasing to the eye.
