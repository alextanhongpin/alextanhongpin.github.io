Basic HTML5 Tutorial

Doctype Declaration

Basic structure of HTML. There is always an opening and closing tag for each elements.

<!DOCTYPE html>
<html lang='en'>
</html>

Like a human, the HTML file also has a head and a body. The indentation makes it easier to read the codes. 

<!DOCTYPE html>
<html lang='en'>
    <head>
	</head>
	<body>
	</body>
</html>

Next, enter the title of the webpage. This will be displayed on the tabs of the browser. The title of your page will be the first thing the search engine will look at.

<!DOCTYPE html>
<html lang='en'>
    <head>
        <title>title goes here</title>
    </head>
    <body>
    </body>
</html>

The rest of the content goes into the body.
