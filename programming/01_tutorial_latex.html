Basic HTML5 Tutorial

Doctype Declaration

Basic structure of HTML. There is always an opening and closing tag for each elements. 

<!DOCTYPE html>
<html lang='en'>
</html>

Like a human, the HTML file also has a head and a body. The indentation makes it easier to read the codes. 

<!DOCTYPE html>
<html lang='en'>
    <head>
	</head>
	<body>
	</body>
</html>

Next, enter the title of the webpage. This will be displayed on the tabs of the browser. The title of your page will be the first thing the search engine will look at. 

<!DOCTYPE html>
<html lang='en'>
    <head>
        <title>title goes here</title>
    </head>
    <body>
    </body>
</html>

The rest of the content goes into the body. There are some other elements that you must know, namely p (paragraph), a (hyperlinks), img (image), h1, h2, h3, h4, h5, h6 (headers, in increasing size), and li (lists). Try this out and see the output. 

<!DOCTYPE html>
<html lang='en'>
    <head>
        <title>title goes here</title>
    </head>
    <body>
	<h1>My first website.</h1>
	<p>Making a website is not as hard as I think it would be!</p>
    </body>
</html>

Now, let's add an image and also a list of motivations. In order to organize your growing website, it is recommended that you keep all your images in a folder called img. This will help keep things neat. Note that the image has no closing tag.

There are two kinds of list, ul (unordered list) and ol (ordered list). The unordered list is basically bullet points, while ordered list are numbered. In between these tags, the list be enclosed in a li tag. 

<!DOCTYPE html>
<html lang='en'>
    <head>
        <title>title goes here</title>
    </head>
    <body>
	<h1>My first website.</h1>
	<p>Making a website is not as hard as I think it would be!</p>
	<img src='img/mypic.jpg'>
	<ul>
		<li>I won't be lazy</li>
		<li>The last thing I want to do is slacking away.</li>
		<li>Alex is a nice person.</li>
	</ul>
    </body>
</html>
